# DATABASE SQL 

## 날짜형식

| 형식| 설명 |
|--------|--------|
|    yyyy    |   네자리연도(숫자)     |
|    YEAR    |   연도(문자)     |
|    MM   |   두자리 값으로 나타낸달     |
|    MONTH   |   달 전체이름     |
|    MON    |   세자리 약어로 나타낸달     |
|    DY    |   세자리 약어로 나타낸 요일     |
|    DAy    |   요일전체     |
|    DD    |   숫자로 나타낸 달의 일     |
|    HH,24HH    |   시간,24시간     |
|    9    |  숫자를 표시     |
|    0    |   0을 강제로 표시     |
|    $   |   부동$기호를 표시     |
|    L    |   부동 지력통화기호 표시     |
|    .   |   소수점 출력     |
|    ,    |   천단위 구분자 출력     |
